# SASInnovate2026
This repo will house the materials for the Pre-Conference Educator Workshop at SAS Innovate 2026. 

Full description below:

## Educators — do you want a sneak peek at exciting new features coming to SAS software?

Then this pre-conference workshop is for you! In an action-packed three hours, we’ll take a hands-on tour of the SAS Viya platform that empowers coders and non-coders alike to deliver data-driven insights — featuring new and enhanced technology such as SAS Viya Copilot, SAS Studio Flows, and AI-generated insights — and how to ensure those insights are ethical and responsible.

## Time: Monday, April 27 from 9am-12pm
## Room: Escondido Room (Vineyard Tower)
